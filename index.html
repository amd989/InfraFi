InfraFi Package Repository

IR WiFi credential receiver daemon for Linux servers with CIR receivers.

Debian / Ubuntu / Mint (apt)

Quick Install

curl -fsSL https://amd989.github.io/InfraFi/setup.sh | sudo bash
sudo apt install infrafid

Manual Setup

curl -fsSL https://amd989.github.io/InfraFi/gpg.key | sudo gpg --dearmor -o /usr/share/keyrings/infrafid.gpg
echo "deb [signed-by=/usr/share/keyrings/infrafid.gpg] https://amd989.github.io/InfraFi stable main" | sudo tee /etc/apt/sources.list.d/infrafid.list
sudo apt update
sudo apt install infrafid

Rocky Linux / RHEL / Fedora (dnf)

Quick Install

curl -fsSL https://amd989.github.io/InfraFi/setup-rpm.sh | sudo bash
sudo dnf install infrafid

Manual Setup

sudo rpm --import https://amd989.github.io/InfraFi/gpg.key
sudo tee /etc/yum.repos.d/infrafid.repo <<EOF
[infrafid]
name=infrafid
baseurl=https://amd989.github.io/InfraFi/rpm/
enabled=1
gpgcheck=1
gpgkey=https://amd989.github.io/InfraFi/gpg.key
EOF
sudo dnf install infrafid
